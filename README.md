# Релиз v1.1

### Обновления:
- Увеличен начальный размер окна
- При выборе варианта его номер пишется в заголовке окна
- Стрелки рисуются красиво и ровно
- Всплывающие окна не закрывают выигрыши
- При задании выигрышей работают кнопки Enter (= OK), стрелка вверх (переход в поле ввода В), стрелка вниз (переход в поле ввода А)
- При появлении окна задания выигрышей фокус курсора сразу ставится на поле ввода А
- Проверка правильности ввода выигрышей в виде заливки и работает сразу после ввода, а не только по кнопке ОК.
